# Database Migrations

## How to Run Migrations

These are manual SQL migrations to run in the Supabase SQL Editor:

1. Go to https://supabase.com/dashboard
2. Select your project
3. Go to SQL Editor
4. Create new query
5. Copy-paste the SQL from the migration file
6. Click "Run"
